# Адрес приложения:

- `http://82.114.226.145:8001`
- `http://82.114.226.145:8001/docs`

## Проверки

Проверка 0. Валидный запрос

Запрос:

```bash
curl -X POST http://82.114.226.145:8001/registration \
  -H 'Content-Type: application/json' \
  -d '{"username":"User2026","email":"user2026@example.com","password":"Strong1!","confirm_password":"Strong1!","age":25}'
```

Ответ:

```json
{"msg":"Пользователь создан","user":"User2026"}
```

Ошибка 1. Возраст меньше 18

Запрос:

```bash
curl -X POST http://82.114.226.145:8001/registration \
  -H 'Content-Type: application/json' \
  -d '{"username":"User2026","email":"user2026@example.com","password":"Strong1!","confirm_password":"Strong1!","age":17}'
```

Ответ:

```json
{"detail":[{"type":"greater_than_equal","loc":["body","age"],"msg":"Input should be greater than or equal to 18","input":17,"ctx":{"ge":18}}]}
```

Ошибка 2. Возраст больше 100

Запрос:

```bash
curl -X POST http://82.114.226.145:8001/registration \
  -H 'Content-Type: application/json' \
  -d '{"username":"User2026","email":"user2026@example.com","password":"Strong1!","confirm_password":"Strong1!","age":101}'
```

Ответ:

```json
{"detail":[{"type":"less_than_equal","loc":["body","age"],"msg":"Input should be less than or equal to 100","input":101,"ctx":{"le":100}}]}
```

Ошибка 3. Пароль `123`

Запрос:

```bash
curl -X POST http://82.114.226.145:8001/registration \
  -H 'Content-Type: application/json' \
  -d '{"username":"User2026","email":"user2026@example.com","password":"123","confirm_password":"123","age":25}'
```

Ответ:

```json
{"detail":[{"type":"value_error","loc":["body","password"],"msg":"Value error, [Пароль] Не может быть '123'","input":"123","ctx":{"error":{}}}]}
```

Ошибка 8. Некорректный username и email

Запрос:

```bash
curl -X POST http://82.114.226.145:8001/registration \
  -H 'Content-Type: application/json' \
  -d '{"username":"Us!","email":"bad-email","password":"Strong1!","confirm_password":"Strong1!","age":25}'
```

Ответ:

```json
{"detail":[{"type":"string_too_short","loc":["body","username"],"msg":"String should have at least 4 characters","input":"Us!","ctx":{"min_length":4}},{"type":"value_error","loc":["body","email"],"msg":"value is not a valid email address: An email address must have an @-sign.","input":"bad-email","ctx":{"reason":"An email address must have an @-sign."}}]}
```
