# Адрес приложения: `http://82.114.226.145:8004/`

Запрос:
`GET http://82.114.226.145:8004/`

Ответ:
`200 OK`

Запрос:
`POST http://82.114.226.145:8004/files/upload`
Заголовок: `X-User-Id: 1`
Файл: `fake.jpg` с текстовым содержимым

Ответ:
`400 Bad Request`

```json
{"detail":"Разрешены только JPEG и PNG, тип проверяется по содержимому файла"}
```

Запрос:
`POST http://82.114.226.145:8004/files/upload`
Заголовок: `X-User-Id: 1`
Файл: `big.png` размером больше `2 МБ`

Ответ:
`413 Request Entity Too Large`

```json
{"detail":"Файл слишком большой. Максимум 2 МБ"}
```

Запрос:
`POST http://82.114.226.145:8004/files/upload`
Заголовок: `X-User-Id: 1`
Файл: корректный `cat.png`

Ответ:
`201 Created`

```json
{"сообщение":"Файл загружен","файл":{"id":4,"имя":"cat.png","размер_байт":68,"тип":"image/png","владелец":"Алиса"}}
```

Запрос:
`GET http://82.114.226.145:8004/files/4/download`
Заголовок: `X-User-Id: 2`

Ответ:
`404 Not Found`

```json
{"detail":"Файл не найден"}
```

Запрос:
`GET http://82.114.226.145:8004/files/4/download`
Заголовок: `X-User-Id: 1`

Ответ:
`200 OK`
Заголовок:
`Content-Disposition: attachment; filename*=UTF-8''cat.png`
